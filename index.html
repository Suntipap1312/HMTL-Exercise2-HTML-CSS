The Basic Languege of the web: HTML

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the web.

In this post,let's focus on HTML. we will learn what is HTML is all about, and why we shold learn it

HTML stands up for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of contents: paragraph, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.
